# StepNip

**Ferramenta de captura e anotação de tela para Windows** — estilo Xnip, com edição flutuante sobre a tela. Por **Cad Capital**.

## ⬇️ Baixar

**[➡️ Baixar a última versão](https://github.com/cadmielbrito/stepnip-download/releases/latest)**

- **StepNip-Setup-x.x.x.exe** — instalador (recomendado).
- **StepNip-Portatil-x.x.x.exe** — versão portátil, roda sem instalar.

Não precisa de .NET nem de administrador. Windows 10/11 64-bit.
Na 1ª execução, se o SmartScreen avisar "Windows protegeu seu PC", clique em **Mais informações → Executar assim mesmo**.

## ✏️ Como usar

Pressione **Shift+Alt+X** para capturar. A barra de ferramentas flutua sobre a tela:
steps numerados com legenda, seta, formas, texto, caneta, recortar, extrair texto (OCR) e captura com rolagem.
